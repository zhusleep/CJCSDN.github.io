总结

方法二和方法三在 push 的时候比较方便。但是在 pull 的时候只能从方法三图中的第一个 url 地址拉取代码。而方法一则不存在这种问题（可能要解决冲突）。所以，如果只进行 push 操作，推荐方法二和方法三，如果也要进行 pull 操作，推荐方法一。
